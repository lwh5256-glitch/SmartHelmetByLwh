# STM32F407 摔倒检测 + 空气质量监测系统

---

## 📋 硬件平台

| 项目 | 说明 |
|------|------|
| **MCU** | STM32F407ZGTx (ARM Cortex-M4, 168MHz) |
| **开发板** | 正点原子 探索者 F407 |

---

## ✨ 功能模块

| 功能 | 状态 | 说明 |
|------|------|------|
| **LCD显示** | ✅ | FSMC驱动，实时显示所有传感器数据 |
| **摔倒检测** | ✅ | 状态机：Normal→FreeFall→Impact→Stable→FALL |
| **PM2.5监测** | ✅ | USART1接收，超标报警（阈值1000μg/m³） |
| **粉尘浓度换算** | ✅ | 自动换算为mg/m³单位显示 |
| **蜂鸣器报警** | ✅ | 摔倒/PM2.5超标双重触发 |
| **按键控制** | ✅ | 4个按键：配网/连接/断开/蜂鸣器开关 |
| **WiFi连接** | ✅ | 支持SmartConfig配网和手动连接 |
| **原子云上报** | ✅ | 每5秒自动上报PM2.5数据（透传模式） |

---

## 📌 引脚配置

| 模块 | 引脚 | 说明 |
|------|------|------|
| **LCD** | FSMC总线 | 由FSMC自动管理 |
| **MPU6050** | PB6(SCL), PB7(SDA) | 软件I2C |
| **PMS7003** | PA10(RX) | USART1，仅接收（9600bps） |
| **ESP8266** | PB10(TX), PB11(RX) | USART3（115200bps） |
| **蜂鸣器** | PF8 | 推挽输出，高电平触发 |
| **KEY0** | PE4 | 下拉，低电平有效 |
| **KEY1** | PE3 | 下拉，低电平有效 |
| **KEY2** | PE2 | 下拉，低电平有效 |
| **KEY_UP** | PA0 | 上拉，高电平有效 |
| **LED2** | - | 心跳LED，每200ms翻转 |


### AT指令
- **连接**：`AT+ATKCLDSTA="设备ID","密码"`
- **断开**：`AT+ATKCLDCLS`

### 数据上报
- **格式**：`PM2.5:xxx\r\n`（透传模式）
- **周期**：每5秒自动上报
- **粉尘浓度换算**：`dust(mg/m³) = PM2.5(μg/m³) / 1000.0`

## ⚠️ 注意事项

1. **lcd_ex.c 不能加入Keil工程**  
   因为 `lcd.c` 第19行已 `#include "lcd_ex.c"`，重复加入会导致符号重定义错误

2. **ESP8266固件要求**  
   需支持原子云AT指令（`AT+ATKCLDSTA`/`AT+ATKCLDCLS`），使用正点原子官方固件

3. **摔倒检测阈值**  
   - FREEFALL阈值：18000  
   - IMPACT阈值：10000  
   - STABLE阈值：4000  
   - 稳定时间：300ms

4. **延时函数**  
   统一使用 `delay_us()`/`delay_ms()`，基于SysTick，168MHz精度

5. **PM2.5报警阈值**  
   当前设置为 1000μg/m³
