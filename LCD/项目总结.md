# STM32F407 智能安全帽项目总结

## 1. 项目定位

本工程基于 `STM32F407ZGTx` 和正点原子探索者 F407 开发板，面向“智能安全帽”场景，当前主线功能为：

- 姿态/摔倒检测：`MPU6050 + 软件 I2C + 摔倒状态机`
- 空气质量监测：`PMS7003 + USART1`
- 网络通信：`ESP8266 + USART3 + 原子云`
- 本地显示：`ATK-MD0240 2.4 寸 SPI LCD`
- 本地告警：`PF8` 蜂鸣器
- 自动补光：`G5516 光敏输入 + LR7843 补光灯开关`

## 3. 当前软件架构

| 路径 | 职责 |
|---|---|
| `Core/Src/main.c` | 系统初始化、主循环、显示刷新、告警协调、WiFi/云端流程 |
| `Core/Src/gpio.c` | Cube/HAL GPIO 初始化，包含光敏、补光、蜂鸣器、LED、软件 I2C 基础 GPIO |
| `Core/Src/usart.c` | `USART1`、`USART3` 初始化和 MSP 配置 |
| `Drivers/HARDWARE/LCD/` | 当前 SPI LCD 驱动，保留 `lcd_*` 兼容接口 |
| `Drivers/HARDWARE/PMS7003/` | PMS7003 数据帧接收与解析 |
| `Drivers/HARDWARE/ESP8266/` | ESP8266 AT 指令、SmartConfig、原子云连接 |
| `Drivers/HARDWARE/FALL_DETECT/` | 摔倒检测状态机 |
| `Drivers/HARDWARE/FILTER/` | 一阶低通和滑动平均滤波 |
| `Drivers/HARDWARE/BEEP/` | 蜂鸣器控制 |
| `Drivers/HARDWARE/KEY/` | 按键初始化与边沿扫描 |
| `MyCode/` | MPU6050、DMP、软件 I2C 支撑代码 |
| `DHT11/` | 历史残留 DHT11 驱动，当前不参与主工程 |

1. 若继续扩展定位功能，新增 `USART6` 初始化、GPS 中断接收和 `RMC/GGA` 解析。
