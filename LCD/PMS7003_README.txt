PMS7003 PM2.5 传感器集成说明
================================================================================

一、说明范围
------------
本文只维护 PMS7003 传感器的硬件连接、串口配置、数据解析、显示和告警逻辑。
整机引脚总表以《硬件连接总表与扩展规划.md》为准。

三、硬件连接
------------
PMS7003 连接到 STM32F407：

| PMS7003 | STM32F407 |
|---|---|
| VCC | 5V |
| GND | GND |
| TX | PA10 (USART1_RX) |
| RX | PA9 (USART1_TX) |
| SET | 拉高或悬空，保持工作模式 |

注意：
- PMS7003 上电后需要约 30 秒预热，预热期间数据可能不稳定。
- PMS7003 为 3.3V TTL 串口电平，PA9/PA10 可直接匹配。

五、串口配置
------------
USART1 当前配置：

| 参数 | 值 |
|---|---|
| 波特率 | 9600 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | 无 |
| 流控 | 无 |
| 接收方式 | 中断接收 |


六、数据处理
------------
当前驱动逻辑：

1. 通过 USART1 中断逐字节接收 PMS7003 数据。
2. 在 `PMS7003_UART_RxCallback()` 中缓存并查找完整 32 字节帧。
3. 校验帧头 `0x42 0x4D` 和校验和。
4. 解析 PM1.0、PM2.5、PM10 及颗粒物数量。
5. `PMS7003_GetData()` 返回当前缓存数据。


七、显示与告警
--------------
当前 `main.c` 中的使用方式：

| 项目 | 逻辑 |
|---|---|
| PM2.5 显示 | `PM2.5:%4d ug/m3` |
| 粉尘换算 | `Dust(mg/m3) = PM2.5(ug/m3) / 1000.0` |
| 报警阈值 | `pm2_5_cf1 > 1000` |
| LCD 告警 | 显示 `PM2.5!!` 或 `PM2.5:%4d ALARM!` |
| 蜂鸣器 | 未摔倒且 PM2.5 超阈值时持续开启 |
| 云端上报 | 原子云连接后约每 5 秒发送 `PM2.5:<value>` |

告警优先级说明：
- 摔倒告警优先级高于 PM2.5 告警。
- WiFi 状态变化后的静默窗口逻辑在 `main.c` 中存在，但当前蜂鸣器出口未使用 `wifi_mute_until_ms` 判断；这与 2026-04-25 日志中的部分描述不完全一致，已在项目日志中标注。


八、数据帧格式
--------------
PMS7003 UART 数据帧固定 32 字节：

| 字节 | 含义 |
|---|---|
| 0-1 | 帧头 `0x42 0x4D` |
| 2-3 | 帧长度 `0x00 0x1C` |
| 4-5 | PM1.0 浓度，CF=1，ug/m3 |
| 6-7 | PM2.5 浓度，CF=1，ug/m3 |
| 8-9 | PM10 浓度，CF=1，ug/m3 |
| 10-11 | PM1.0 浓度，大气环境 |
| 12-13 | PM2.5 浓度，大气环境 |
| 14-15 | PM10 浓度，大气环境 |
| 16-27 | 不同粒径颗粒数量 |
| 28 | 固件版本 |
| 29 | 错误代码 |
| 30-31 | 校验和 |

所有 16 位字段均为大端序。


九、测试建议
------------
1. 上电后等待约 30 秒。
2. 观察 LCD 上 PM2.5 是否稳定刷新。
3. 使用安全可控的颗粒物源进行短时测试，确认数值变化。
4. 超阈值测试时注意传感器量程，长期高浓度测试可能污染气道。
